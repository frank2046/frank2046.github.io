# 坦克大战（浏览器版）

一个纯前端（HTML/CSS/JS）的坦克大战小游戏：Canvas 渲染、砖墙可打碎、钢墙不可破坏、敌人 AI、胜负与计分。

## 如何运行（Windows 最简单）

由于你的环境里看起来没有 `python` / `node`，**直接双击打开 `index.html`** 即可在浏览器运行（本项目不依赖后端/接口/资源加载）。

如果你更喜欢用本地服务器，也可以用任意静态服务器工具把目录 `d:\ai\TestProject` 作为站点根目录启动。

## 操作

- **W / S**：前进 / 后退
- **A / D**：平移（小幅）
- **← / →**：转向
- **Space**：开火
- **P**：暂停 / 继续
- **R**：重新开始

## 规则

- **砖墙**：可被炮弹击碎（需要 2 发）
- **钢墙**：不可破坏
- 消灭全部敌人即胜利；生命为 0 则失败
